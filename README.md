# 호주 은퇴 재정 도구 (Australian Retirement Toolkit)

브라우저에서 바로 쓰는 호주 은퇴 재정 계산기 모음. 가입·설치 불필요, 입력값은 브라우저 안에서만 계산되며 서버로 전송·저장되지 않습니다.

**바로 사용:** https://hojuorgo.github.io/age-pension-calculator/

## 포함 도구

- **노령연금 계산기** — Age Pension 수령액을 소득·자산 테스트로 계산 (Work Bonus 포함, 2026년 3월 기준 요율)
- **포트폴리오 배당 계산기** — 총자산 → Super/Non-Super 플랫폼 → 자산분류별(커버드콜·배당주·개별주·지수·현금) 비중·배당율 → 월/연 배당 현금흐름
- **인출 백테스트 시뮬레이터** — QLD·SCHD·JEPQ·VAS 배당조정종가 실데이터(2006~) 기반. 자가배당(매도 인출) / 배당 현금흐름 두 모드, 롤링 백테스트로 고갈확률·P10·MDD 산출

## 면책

교육·참고용 계산기이며 개인 맞춤 금융조언이 아닙니다. 실제 결정은 등록된 재무설계사나 Centrelink·ATO 등 공식 기관에 확인하세요. 요율은 작성 시점 기준이며 변경될 수 있고, 과거 수익률은 미래를 보장하지 않습니다.
